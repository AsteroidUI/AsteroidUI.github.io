Asteroid Desktop

A free tool that allows you to do basic management of vctl container engine on macOS for running containers and Kubernetes clusters.

Easy&Access

Handy management of vctl container engine through UI & Touch Bar.
Configure your virtual machines for containers and Kubernetes cluster without CLI.

Light & Free

It's light and it's free.

How to Install

Asteroid Desktop requires macOS 10.15 to run.

Download and unzip the latest release: v1.0.0 (1460)
Asteroid Desktop.app is signed with the Apple Development certificate, so to allow Opening a Mac app from an unidentified developer is required on the first run.
